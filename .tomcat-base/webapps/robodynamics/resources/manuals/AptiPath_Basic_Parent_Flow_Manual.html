AptiPath Career Discover Basic - Parent Flow Manual

This manual explains the complete parent-first flow from plan purchase to report usage.

Start Basic Plan

1. What parents see first

  • Career Discover is shown as an independent module.
  • Basic and Pro pricing is clear with scope differences.
  • Primary CTA is Basic plan purchase for first-time users.
  • Existing users can login and continue from dashboard.

2. 4-step conversion journey

  1. Parent starts Basic plan checkout.
  2. Parent logs in or registers parent + student profile.
  3. Parent questionnaire is completed first.
  4. Student intake is completed, then student starts test.

3. Parent questionnaire scope

  • Academic goals, budget range, and timeline.
  • Observed strengths and areas of concern.
  • Expected support style from mentors and platform.
  • Location and coaching constraints.

4. Student intake scope

  • Interests, hobbies, dislikes, and motivation.
  • Self-rated comfort by subject and learning mode.
  • Fears, pressure points, and achievement signals.
  • Readiness check before test starts.

5. Basic vs Pro positioning

  • Basic: starter diagnosis and direction snapshot.
  • Pro: deeper competency, roadmap, and action planning.
  • Recommended to upgrade to Pro after Basic completion.

6. Parent questions to answer clearly

  • How scores are calculated and validated.
  • Which responses affected each key output metric.
  • What confidence level the system has in recommendations.
  • What additional assessment is needed before high-stakes decisions.
