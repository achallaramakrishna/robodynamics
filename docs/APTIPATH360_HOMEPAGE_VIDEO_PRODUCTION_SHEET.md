# AptiPath360 Homepage Intro Video - Production Sheet

## 1) Master Specs
- Duration: `45 seconds`
- Primary format: `1920x1080 (16:9)` for homepage hero
- Secondary cut: `1080x1920 (9:16)` for reels/shorts
- Frame rate: `30 fps`
- Audio loudness target: `-14 LUFS`
- Style: clean futuristic, parent-trust, premium education-tech

## 2) Core Message
- Problem: parents and students face career confusion.
- Solution: AptiPath360 gives structured, adaptive, stage-aware career discovery.
- Outcome: actionable report with clear next steps.
- CTA: Start now + support number/WhatsApp.
