# AptiPath Product Vision and Execution Plan (2026-02-23)

## 1. Product Intent
AptiPath should be positioned as a parent-first AI career guidance product, not as a mixed list of unrelated modules.

Core intent:
- Give parents clarity in 3-4 clicks.
- Build trust through transparent scoring and clear evidence.
- Start with low-risk entry (Basic), then expand to deeper planning (Pro / Phase B).

## 2. Module Separation (Brand Architecture)
The platform should present three independent modules:

1. Career Discover (AptiPath)
2. Exam Prep
3. Tuition

Rules:
- Do not mix pricing narratives across these modules on the same conversion screen.
- Use Career Discover as the parent onboarding entry module.
- Exam Prep and Tuition remain visible as independent next modules.

## 3. Parent-First Conversion Flow (Target: 3-4 Clicks)
Primary conversion flow:

1. Parent selects Career Discover Basic.
2. Parent logs in (existing) or registers (new).
3. Parent completes questionnaire first.
4. Student completes intake and starts test.

Business intent:
- Keep first-time decision friction low.
- Create trust before asking for high-commitment upgrades.
- Ensure data quality before report generation.

## 4. Pricing and Packaging
Career Discover packaging:

- Basic: Rs 599 + GST (one-time)
- Pro: Rs 1299 + GST (one-time)

Differentiation:
- Basic: starter diagnosis, direction snapshot, key signals.
- Pro (Phase B): deeper competency mapping, roadmap, state/national path options, and parent strategy support.

Upgrade rule:
- Pro should be suggested after Basic completion to use real student evidence.

## 5. Data Capture Before Guidance
### Parent questionnaire
- Goals, timeline, budget constraints
- Support expectations
- Observed strengths and concerns
- Locality and coaching constraints

### Student intake
- Interests, dislikes, motivation
- Subject comfort and confidence
- Achievements and fears
- Learning style and aspiration signals

Outcome:
- Better personalization
- Better authenticity in reports
- Better parent acceptance of recommendations

## 6. Report Authenticity Requirements
Every report must answer:

- How was the score computed?
- Which questions contributed most?
- What is confidence level?
- Where is uncertainty high?
- Why does fit index differ from raw score?

Recommended additions:
- Question contribution table (top positive / negative contributors)
- Confidence band label (low/medium/high confidence)
- Validity checks (attempt quality, speed anomalies, blank patterns)
- Clear disclaimer for high-stakes outcomes (IIT/NEET/CAT are long-horizon predictions)

## 7. Pedagogy Layer Expansion
Current gap: pedagogy and learning-evidence layer is thin.

Near-term upgrades:
- Add subject foundation blocks (Math, Biology, Physics, Language)
- Add reasoning and general awareness calibration
- Add story/composition response for thought process signals
- Add progression logic by grade (8/9/10)

## 8. Phase B Product Scope (Pro Plan)
After Basic completion, Pro expands to:

- Career search with filter-by-interest and grade
- State/national competitive exam map
- Last-year cutoff ranges and fee bands
- Government vs private comparison
- College ranking framework (academics, affordability, outcomes, location fit)
- Coaching center discovery by state/city with trust signals
- AI counselor Q&A interaction layer

## 9. Benchmarking Framework (Where to Evaluate Quality)
Benchmark dimensions:

1. Conversion quality
- Homepage-to-checkout CTR
- Registration completion rate
- Parent questionnaire completion rate

2. Diagnostic quality
- Internal consistency of scores
- Parent trust score on report clarity
- Recommendation acceptance rate

3. Pedagogical quality
- Roadmap usefulness rating
- Action plan follow-through rate
- Repeat engagement in 30/60 days

4. Business quality
- Basic to Pro upgrade rate
- CAC payback period
- Refund and complaint rate

## 10. Execution Plan (Practical Rollout)
### Phase 0: Completed in this release
- Homepage cleaned and restructured with independent module cards.
- Career Discover Basic branding made primary entry path.
- Basic/Pro Career pricing updated to Rs 599 / Rs 1299 in checkout plan configuration.
- Parent flow manual added for transparency.

### Phase 1: Conversion hardening (next)
- Step wizard for parent questionnaire.
- Step wizard for student intake.
- Progress indicator for 4-step flow.
- Explicit existing-user shortcut path.

### Phase 2: Report validity hardening
- Add question-to-score contribution view.
- Add confidence and validity signals.
- Add section-weight explanation panel.
- Add “why this recommendation” narrative.

### Phase 3: Phase B Pro rollout
- Career search engine
- Exam/college roadmap
- Cost and cutoff intelligence
- State/city guidance expansion

### Phase 4: Continuous quality loop
- Parent feedback loops after report read
- Roadmap outcome tracking
- Model recalibration with observed outcomes

## 11. Non-Negotiables
- Parent-first clarity before payment
- Independent module branding
- Transparent scoring logic
- No over-claiming IIT/NEET/CAT certainty from one short test
- Evidence-backed recommendations only
