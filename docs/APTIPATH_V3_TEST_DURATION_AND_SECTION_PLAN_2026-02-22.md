# AptiPath 360 v3 Test Duration and Section Plan (2026-02-22)

## Test Duration
- Student adaptive test duration: **78 to 92 minutes**

## Question Design
- Bank size target: **294 questions** (`v3` pool)
- Questions delivered per student attempt: **66 adaptive questions**
- Baseline routing: **14 anchor questions** (2 per section)
- Dynamic routing: remaining questions adapt by:

  - confidence level
  - response time
  - section coverage gaps
