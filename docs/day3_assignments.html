15 Java Full Stack Assignments

Designed to reinforce Day 1-3 topics (classes, inheritance, interfaces, static members, exception handling, collections, generics, properties). Each assignment includes a clean deliverable, acceptance hints, and related keywords for Git submissions.

  1. Student Profile Builder

    Encapsulate student data with private fields, validation, and factory methods. Deliver getter-only APIs plus `equals`/`hashCode` overrides.

    classesencapsulationvalue object

  2. Account Hierarchy

    Create abstract `Account` with `SavingsAccount` and `CheckingAccount` subclasses. Override monthly charges and provide `super` calls where needed.

    inheritanceabstract classsuper

  3. Reportable Interface Chain

    Define `Reportable` interface with default headers and static helpers. Implement multiple classes to print CSV or Markdown reports.

    interfacesdefault methodsstatic helper

  4. Singleton Configuration

    Use a `final` class with a private constructor and static factory to expose immutable `Properties`. Add lazy loading with `synchronized` block.

    staticsingletonproperties

  5. Exception-Safe DAO

    Create a DAO method that throws custom `DataReadException` and uses multi-catch to handle `SQLException` and `IOException`. Wrap and rethrow while logging.

    exception handlingmulti-catchcustom exception

  6. Try-With-Resources Logger

    Implement `LogWriter` with `AutoCloseable`. Use try-with-resources to append log entries and ensure file handles close.

    try-with-resourcesAutoCloseableI/O

  7. Leaderboard with Comparable

    Model `StudentScore` implementing `Comparable` by GPA. Sort a list using `Collections.sort` and a `Comparator` for name tiebreakers.

    collectionscomparatorssorting

  8. Collection Cleanup Service

    Write a service that removes duplicates using `HashSet`, keeps order with `LinkedHashSet`, and sorts with `TreeSet`. Demonstrate iteration extremes.

    sethashingiteration

  9. Properties Reader Util

    Build a utility reading `config.properties`, exposing defaults via `getProperty(key, defaultValue)`, and watching for reloads.

    propertiesIOAPIs

  10. Generic Pair & Stack

    Create `Pair` and `Stack` classes. Include methods demonstrating bounded type parameters (``).

    genericsbounded typesdata structures

  11. Comparator Chain Lab

    Sort a collection of `Course` records first by difficulty, then by enrollment using chained comparators via `thenComparing`.

    comparatorchainingsorting

  12. Stateful Enum Config

    Define an enum `Environment` with `STAGING/PROD`, each mapping to property files. Provide static helper for `isProduction()` checks.

    enumstaticconfig

  13. Debug Logging Challenge

    Deliver multi-catch block documented in Git with known issues and tests that reproduce `ArrayIndexOutOfBounds` and `NullPointerException` scenarios, plus fixes.

    debugloggingtests

  14. Output Prediction Exercise

    Submit a short write-up analyzing Falcon `finally` + `return`, iterator mutation, and wildcard generics output puzzles from the session. Capture answers in README.

    outputexplaincode reasoning

  15. Full-Stack Demo Mini App

    Compose a Spring boot controller returning JSON from collection utilities, handling exceptions, and injecting config via `@Value` from properties.

    SpringRESTexception

  16. Beta Test Plan

    Document how new parent/student logins should be created, how to reset passwords, and how to skip payment integration during beta with steps and sample API calls.

    betauser flowdocumentation
Deliverables: create Git branches per assignment, describe the workflow, include unit tests where applicable, and push with descriptive commit messages before the next sync.
