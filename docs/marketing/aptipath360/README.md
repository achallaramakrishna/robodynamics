# AptiPath360 Marketing Kit

This folder contains ready-to-use sales and marketing documents for AptiPath360.

## Files

1. `AptiPath360_One_Page_Brochure.md`
2. `AptiPath360_Sales_Deck_Script.md`
3. `AptiPath360_Proposal_Template.md`
4. `AptiPath360_Objection_Handling.md`
5. `AptiPath360_Outreach_Templates.md`
6. `AptiPath360_10_Slide_Presentation.md`
7. `pdf/AptiPath360_Brochure_Print.html`
8. `pdf/AptiPath360_Proposal_Print.html`
9. `variants/AptiPath360_School_Variant.md`
10. `variants/AptiPath360_Coaching_Variant.md`
11. `variants/AptiPath360_Parent_Direct_Variant.md`

## How to use

1. Start with the one-page brochure for first conversation.
2. Use the sales deck script for school/coaching demos.
3. Send the proposal template after interest is confirmed.
4. Use objection handling during calls and follow-ups.
5. Use outreach templates for WhatsApp, email, and call opening.
6. Use the 10-slide file to build your PPT quickly.
7. Open the print HTML files in browser and print as PDF.
